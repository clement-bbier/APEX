# Decision Template

Copy this template for each new decision entry in `DECISIONS.md`.

---

## DNNN — Decision Title (YYYY-MM-DD)

| Field | Value |
|---|---|
| Date | YYYY-MM-DD |
| Session | NNN |
| Decision | One-line description of the decision |
| Status | PROPOSED / ACCEPTED / SUPERSEDED |

### Context

What situation prompted this decision?

### Alternatives Considered

1. **Alternative A**: Description. Pros/cons.
2. **Alternative B**: Description. Pros/cons.
3. **Alternative C (chosen)**: Description. Why chosen.

### Justification

Why was this alternative selected? What evidence supports it?

### References

- Academic papers, ADRs, or external sources that informed the decision.

### Consequences

What changes does this decision imply? What must be done differently?
