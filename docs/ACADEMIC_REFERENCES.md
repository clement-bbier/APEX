# APEX Academic References

Centralized bibliography for all academic references cited across APEX.

**Last updated:** 2026-04-12

## How to use this document

- When implementing a feature or method, cite the canonical reference inline in the docstring
- When adding a new reference, follow the format: Author (Year). "Title." *Journal*, Vol(Issue), Pages. DOI
- Only Tier-1 sources accepted (see criteria at bottom)
- References are numbered within each section for easy citation

Econometrics, Journal of Portfolio Management, Econometrica, Journal of Risk,
Quantitative Finance, Journal of Financial Econometrics, Machine Learning.

**University press books:**
MIT Press, Cambridge University Press, Princeton University Press, Oxford University
Press, Stanford University Press, Springer (academic series only).

**Working papers (conditional):**
SSRN / arXiv q-fin from Tier-1 institutions only (MIT, Princeton, Stanford, Chicago,
NYU, Imperial College, ETH Zurich, AQR, Two Sigma, Citadel, D.E. Shaw).

## Sources FORBIDDEN

- Blog posts not peer-reviewed
- YouTube tutorials
- Medium articles
- Reddit / StackOverflow (except justified exceptions with explicit note)
- Vendor whitepapers without academic citations
- Self-published books without university press backing
